My Time in Malaysia

Absolute Value Function
Transformations

I chose this photo because the Petronas Twin Towers are an iconic landmark of Malaysia and I find them very interesting. This is a photo I took during my first visit there.

To model the sign appearing in the photo, I first determined that its shape is very similar to an absolute value function, so I chose the parent function y = |x| as my starting point.

By observing the direction of the angle formed by the two rays in the image, I noticed that it opens downward rather than upward. This indicates a reflection across the x-axis, so I modified the function to y = -|x| to match the orientation of the graph.
