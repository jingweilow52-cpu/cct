My Time in Malaysia

Absolute Value Function
Transformations

I chose this photo because the Petronas Twin Towers are an iconic landmark of Malaysia and I find them very interesting.

To model the sign appearing in the photo, I first determined that its shape is very similar to an absolute value function, so I chose the parent function y = |x| as my starting point.

By observing the direction of the angle formed by the two rays in the image, I noticed that it opens downward rather than upward. This indicates a reflection across the x-axis, so I modified the function to y = -|x|.

I then applied a vertical stretch with coefficient −1.6.

The expression 0.9(x+3.9) shifts the graph 3.9 units left and creates a slight horizontal stretch.

Finally, I translated the graph upward by 15.8 units.

f(x)=−1.6|0.9(x+3.9)|+15.8

Domain and Range

Domain: D:{x∈R | −8≤x≤0}

Range: R:{y∈R | 9.90≤y≤15.80}

Secant Function
Transformations

I chose this photo because it captures one of my memorable experiences in Malaysia.

I modeled the curve using the parent function y = sec(x).

The graph opens downward, so I reflected it across the x-axis.

I applied a vertical stretch by a factor of 4.6.

The expression 1.195(x−0.865) compresses the graph horizontally and shifts it right by 0.865 units.

Finally, I translated the graph upward by 5.8 units.

f(x)=−4.6sec(1.195(x−0.865))+5.8

Domain and Range

Domain: D:{x∈R | 0.6≤x≤1.05}

Range: R:{y∈R | 0.96≤y≤1.20}
